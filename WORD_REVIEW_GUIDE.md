# 單字庫審核規範

本文件定義 VocabTrainer 單字庫的收錄標準與審核流程。

---

## 一、不適合收錄的單字類型

以下類型的單字**必須從資料庫中移除**：

### 1. 色情 / 成人內容
- 色情網站名稱（如 xnxx、pornhub 等）
- 性行為相關粗俗詞彙
- 成人產業術語

### 2. 腥羶色內容
- 過於暴力、血腥的詞彙
- 仇恨性語言、種族歧視用詞
- 煽動性極端用語

### 3. 不雅俚語 / 粗口
- 英文粗口（f-word、s-word 等）
- 具侮辱性的稱呼

### 4. 非英文單字
- 網路用語縮寫、亂碼
- 品牌名、人名（除非有明確教育意義）
- 數字、符號組合

---

## 二、適合收錄的單字標準

- 收錄於牛津 5000 / CEFR A1–C2 詞彙表的正式英文單字
- 學術用途、日常用語、商業英語等正規詞彙
- 有明確、正確的中文定義

---

## 三、定義品質標準

### 好的定義
- **簡潔**：20 字以內能說清楚
- **準確**：符合單字最常用的意思
- **易懂**：使用常見中文詞彙，避免過度文言

### 不好的定義（需替換）
- 機器翻譯腔調過重（如「矛盾；抵觸；反駁的行為」→ 應改為「矛盾」）
- 定義過長、包含太多次要含義
- 定義內容錯誤

---

## 四、用戶建議定義審核流程

1. 用戶在測驗時可透過「✏️ 建議定義」按鈕提交更好的定義
2. 建議會存入 `words.suggested_meaning` 欄位
3. 定期執行 `review_suggestions.py` 使用 Claude API 自動審核：
   - 建議正確且更易懂 → 替換 `meaning`
   - 建議錯誤或不適合 → 清除建議，保留原始定義
4. Admin 可在管理介面直接搜尋 / 刪除不適合的單字

---

## 五、Admin 手動審核指令

```bash
# 預覽待審核建議（不執行任何變更）
python3 review_suggestions.py --dry-run

# 正式執行審核並同步到 NAS
python3 review_suggestions.py
```

---

## 六、定期維護建議

- 每週執行一次 `review_suggestions.py`
- 有用戶回報問題時立即在 Admin 介面處理
- 每季重新掃描一次單字庫，移除不適合的詞彙
